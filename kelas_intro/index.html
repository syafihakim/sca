Pembelajaran di Universiti serba laju kerana, pelajar dianggap "dah boleh berdikari"

Di Universiti, pelajar dianggap :

  • Telah pandai berfikir tentang logik coding
  • Mampu untuk menggabungkan setiap topik dengan sendiri
  • Tahu untuk berdikari apabila membuat coding
  • Tahu apa yang perlu dilakukan apabila tersangkut ketika membuat coding
  • Tahu menyelesaikan ralat yang berlaku dalam coding sendiri

Tapi.. majoriti pelajar tak pernah sentuh coding langsung sebelum mula semester. Jadi, mereka tercicir dan tertinggal.
