Kenapa Ramai Pelajar Tak Faham Coding?

Pada semester 1, pelajar akan jumpa subject :

  • Programming Fundamentals
  • Programming I
  • atau nama-nama lain

dengan harapan mereka akan belajar dari zero.

Awal-awal macam boleh faham

Semua yang lecturer ajar "makes sense" je

Tapi rupanya ada masalah tersembunyi

Yang akan menghantui pelajar hingga hujung semester

Sebab bila dapat soalan coding je.. ramai pelajar tiba-tiba rasa blur.

Bila tengok soalan, mereka fikir:

“Nak kena buat apa dulu ni??”

“Dalam kelas tadi macam faham, tapi tak boleh pun buat soalan sendiri”

Mereka tahu syntax (cara tulis coding),

Mereka boleh sebut keyword-keyword yang diorang belajar macam : cin, for loop, class, private, void, etc

tetapi tak tahu cara berfikir untuk selesaikan soalan

Tak kisah anda belajar apa sekali pun

Topik-topik semester 1 :

  • If-Else
  • Loops
  • Functions
  • Array

Atau subject semester 2 dan seterusnya :

  • Object Oriented Programming
  • Data Structure and Algorithms
  • Programming II

Sebab Itu Coding Mula Rasa Susah

Permulaan yang tak jelas, habis semuanya hancur

  • Makin lama makin keliru
  • Topik baru makin susah difahami
  • Mula rasa coding terlalu susah (not for me)

Akhirnya tak faham pun diorang sebenarnya belajar apa

Belajar takat nak habiskan semester je 😥

Tetapi Sebenarnya Masalahnya Bukan Anda

Masalahnya ialah…

Ramai pelajar tak pernah nampak bagaimana coding sebenarnya berfungsi dari awal.
