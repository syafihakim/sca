Sebelum Workshop

2–3 hari sebelum workshop, anda akan dimasukkan ke dalam group.

Anda akan mula dengan:

  • Tutorial latihan menulis coding
  • Latihan asas untuk biasakan diri

Fokus awal: C++ sahaja untuk asas yang jelas.

🎯 Goal: Selesa menulis coding

Hari Pertama (1 Jam 30 Minit)

Apa anda akan belajar

  • • Cara buat coding dari A-Z
  • • Cara cari formula yang tepat
  • • Cara membaca & membaiki error
  • • Cara tentukan program betul atau tidak

Apa akan berlaku dalam kelas

  • • Anda terus buat coding secara LIVE
  • • Dibimbing langkah demi langkah
  • • Jika tersangkut → breakout room (bantuan personal)
🎯 Goal: Mendedahkan semua asas penting

Selepas Hari Pertama

  • Lebih banyak soalan Easy diberikan
  • Teruskan latihan sendiri
  • Share progress dalam group
  • Tanya jika tersangkut

Di sinilah momentum dibina — bukan sekadar faham, tapi mula buat sendiri.

Hari Kedua (1 Jam 30 Minit)

  • Menjawab soalan lebih mencabar
  • Gunakan teknik sama untuk pelbagai language
  • Langkah seterusnya selepas workshop
  • Jemputan sambung ke kelas 1-to-1
🎯 Goal: Mula berdikari dalam coding

Selepas Workshop

  • Soalan Medium–Hard diberikan
  • Teruskan latihan secara konsisten
  • Share progress & tanya soalan

Group akan dibubarkan selepas beberapa hari.

Namun, jika masih perlukan bantuan, anda boleh contact saya secara personal.
