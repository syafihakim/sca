Program Persediaan Asas Coding (CS/IT/S.E)

Dalam semester 1 nanti, para pelajar kursus-kursus di atas akan diperkenalkan dengan subjek coding.

Namun, para pelajar yang :

  • tidak mempunyai asas
  • pertama kali mendengar tentang “coding”
  • berharap University/Kolej akan bimbing mereka dari awal

berkemungkinan besar akan tercicir, berasa stress dan tak boleh memahami subjek coding – dari awal semester hingga tamat semester!

*Berdasarkan pengalaman kami sejak 2017 lagi

  • Langkah seterusnya untuk memulakan program

Tentang program ini

Ringkasan program

Program persediaan berstruktur secara individu (1-to-1) untuk pelajar yang bercadang menguasai asas subjek coding.

Asas akan dibina dari kosong, jadi pelajar yang tiada pengalaman langsung pun tetap sesuai! Bahkan, program ini dibuat khas untuk golongan tersebut.

Kisah kewujudan program ini

Sejak 2018, saya (Sir Syafi) telah mula menjadi mentor dan tutor kepada junior-junior coding. Sehingga tahun ini (2026) dan setelah mengajar dan guide beratus ratus pelajar coding di seluruh Malaysia, saya nampak satu pattern yang sama.

Pelajar yang memasuki Degree/Diploma/Kolej tanpa asas coding dan pertama kali belajar coding di sana memang akan :

  • tercicir
  • tak faham pun apa yang dipelajari setiap kelas
  • semakin lama dalam semester semakin tak faham (kerana setiap topik memang berkait!)
  • tidak mampu pun nak buat soalan-soalan level Easy (yang sepatutnya boleh dibuat selepas belajar coding selama 15 minit!)
  • tidak memahami maksud coding sebenar dan tak tahu apa sebenarnya yang mereka pelajari

dan macam-macam lagi rintangan yang sepatutnya boleh dielakkan.

Maka, pada tahun 2026 saya berhasrat untuk bertukar arah dan menjadikan institusi ini sebagai pusat persediaan awal pelajar-pelajar coding.

Matlamat Akhir Program

Melalui sesi Live secara 1-to-1, pelajar akan :

  • Mampu menerangkan "apa itu coding"
  • Mampu membaca setiap baris coding yang ditulis sendiri atau dari Internet/buku
  • Mampu menerangkan cara coding berfungsi dengan analogi mudah
  • Mampu menyiapkan coding dari awal hingga habis bagi peringkat asas
  • Mampu untuk menentukan sama ada code yang ditulis dikira tepat atau tak
  • Mampu untuk membaca error dan membaiki error dengan independent (berdikari)

Ringkasannya, pelajar yang berjaya menamatkan program persediaan ini akan mempunyai ilmu asas coding setara dengan pelajar coding di University/Kolej yang telah belajar coding selama 1 bulan (bahkan lebih dari itu kerana teknik dan silibus Syafi Coding Academy yang sangat mantap)

Bagaimana pelajar akan menguasai asas

Sesi 1-to-1 : Hanya anda dan saya, 100% attention saya untuk anda

  • Saya ajar theory untuk setiap topik
  • Saya ajar step-by-step untuk guna topik tu dalam apa-apa soalan
  • Saya ajar anda cara berfikir “waktu bila nak guna topik ni?”

Dalam sesi macam ni, saya ajar melalui :

  • Saya explain secara visual (bentuk lukisan)
  • Saya tunjuk cara buat coding satu per satu

Tanya apa jua soalan dan pertanyaan yang terlintas di fikiran

Guided coding session with Sir Syafi (No blind practice)

Sepanjang sesi 1-to-1 nanti, kita juga akan :

  • Buat coding bersama-sama
  • Anda dapat lihat output sebenar
  • Anda dapat lihat error sebenar

Saya guide anda step-by-step sehingga siap

Saya tak suap anda jawapan. Saya sentiasa menunggu kata-kata "Tak tahu la sir" untuk saya bawa dan pimpin anda ke jawapannya.

Saya akan cetus idea anda dan pimpin anda sehingga habis topik tersebut.

Struktur dan tempoh program

Struktur program

Program ini dijalankan secara 1-to-1 dan live, sepenuhnya antara pelajar dan Sir Syafi.

  • Tiada kelas berkumpulan
  • Tiada tutorial rakaman untuk bahagian penting
  • Setiap sesi disesuaikan mengikut tahap dan keperluan pelajar

Pembelajaran dijalankan secara berperingkat:

  • bermula dari asas paling awal
  • diikuti penerangan dan latihan
  • difokuskan kepada kefahaman sebelum beralih ke topik seterusnya

Fokus utama program ini ialah keupayaan pelajar untuk memahami dan menggunakan konsep, bukan sekadar menghabiskan silibus atau mengikut jadual tetap.

Penggunaan sesi live dan rakaman sokongan

Untuk memastikan sesi live digunakan secara paling efektif, pembelajaran dijalankan melalui gabungan sesi live dan bahan sokongan.

  • Sesi live digunakan untuk penerangan, latihan, perbincangan, dan bimbingan cara berfikir
  • Rakaman pendek disediakan untuk penerangan tertentu yang sesuai diulang tonton

Rakaman ini bertujuan sebagai rujukan dan pengukuhan, supaya pelajar boleh mengulang bahagian tertentu mengikut keperluan masing-masing, tanpa menjejaskan fokus utama sesi live.

Tempoh program

Dari segi tempoh, program ini kebiasaannya mengambil masa sekitar 3 hingga 4 minggu.

Namun, tempoh sebenar bergantung kepada:

  • tahap permulaan pelajar
  • kadar kefahaman
  • penglibatan sepanjang sesi

Program ini tidak ditentukan oleh bilangan kelas semata-mata, tetapi berdasarkan pencapaian asas yang diperlukan sebelum pelajar memasuki pengajian Diploma atau Ijazah.

Apa yang disediakan sepanjang program

Sepanjang program ini, pelajar akan dibekalkan dengan bahan sokongan pembelajaran.

  • nota ringkas
  • bahan rujukan
  • panduan untuk ulangkaji dan pengukuhan

Semua bahan disediakan sebagai alat sokongan pembelajaran, dan digunakan bersama sesi bimbingan secara langsung — bukan sebagai pengganti.

Bila program dianggap tamat

Program ini dianggap tamat apabila pelajar berjaya melalui satu sesi demonstrasi kefahaman secara praktikal.

Dalam sesi ini, pelajar akan diminta untuk menyiapkan satu tugasan coding peringkat asas yang menggabungkan beberapa konsep utama yang telah dipelajari — dari awal hingga kod tersebut berfungsi dengan betul.

Sepanjang tugasan ini, pelajar:

  • menunjukkan kefahaman terhadap konsep yang digunakan
  • mampu membaca dan menerangkan kod yang ditulis
  • mampu membaiki kesilapan dan menyiapkan tugasan secara berdikari

Pelajar dibenarkan merujuk nota, bahan rujukan, dan carian Internet semasa sesi ini — kerana inilah cara coding dilakukan dalam dunia sebenar.

Tiada jawapan tunggal yang “betul”. Selagi kod berfungsi dan pelajar memahami apa yang dilakukan, tugasan tersebut dianggap berjaya.

Penilaian dibuat berdasarkan keupayaan sebenar pelajar semasa sesi, melalui pemerhatian dan perbincangan, bukan melalui peperiksaan bertulis.

Yuran & pengambilan pelajar

Bagaimana yuran program ditentukan

Yuran program ini ditentukan berdasarkan sifat pembelajaran yang dijalankan secara individu (1-to-1), serta penglibatan langsung Sir Syafi sepanjang program berlangsung.

Fokus utama program ini bukan pada bilangan sesi atau tempoh semata-mata, tetapi pada keupayaan pelajar untuk mencapai tahap kefahaman asas yang diperlukan sebelum memasuki pengajian Diploma atau Ijazah.

Setiap pelajar dibimbing secara peribadi, dengan pendekatan yang disesuaikan mengikut tahap, kadar pembelajaran, dan keperluan masing-masing.

Yuran program

Yuran bagi Program Persediaan Asas Coding ini adalah sebanyak RM500.

Yuran ini merangkumi keseluruhan program persediaan, termasuk sesi bimbingan secara live, bahan sokongan pembelajaran, serta penilaian kefahaman pelajar berdasarkan keupayaan sebenar.

Kapasiti & pengambilan pelajar

Oleh kerana program ini dijalankan secara 1-to-1, kapasiti pengambilan pelajar adalah terhad pada satu-satu masa.

Pengambilan pelajar dibuat berdasarkan ketersediaan masa dan kesesuaian keperluan pelajar dengan objektif program ini. Tidak semua pelajar yang membuat pertanyaan akan diterima serta-merta.

Siapa yang sesuai mengikuti program ini

Program ini sesuai untuk pelajar yang akan memasuki pengajian Diploma atau Ijazah dalam bidang CS / IT / Software Engineering, dan ingin membina asas coding dengan lebih teratur dan yakin.

Program ini kurang sesuai untuk pelajar yang hanya mencari kelas ringkas tanpa komitmen, atau yang tidak bersedia untuk terlibat secara aktif dalam sesi pembelajaran.
