Tentang Program Ini

Direka khas untuk pelajar yang tak pernah mengenali coding. Pelajar lepasan PPAC ini bakal berada 10 langkah di hadapan rakan-rakan pada semester 1 nanti. Lebih yakin dan bersedia.

Target Pelajar

Lepasan SPM/STPM/Diploma yang bakal mengambil jurusan yang mempunyai subjek coding

Format Program

Sesi 1-to-1 dan latihan praktikal menggunakan silibus yang dicipta khas bagi mempercepatkan penguasaan coding

Tempoh

Lebih kurang 3-4 minggu (flexible scheduling).

Mengapa PPAC diwujudkan

Sejak 2018, saya telah membimbing ratusan pelajar coding di seluruh Malaysia dan mendapati satu pattern yang sama berulang â€” ramai pelajar yang memasuki pengajian Diploma atau Ijazah tanpa asas coding sering :

  • Tercicir dan tak faham apa yang dipelajari dalam kelas
  • Tersangkut dari awal semester hingga akhir semester
  • Tak mampu membuat soalan yang sepatutnya cuma mengambil masa 10 minit sahaja
  • Berada dalam keadaan seolah-olah tidak belajar coding (selepas 6 bulan di Universiti/Kolej)


Maka pada 2026 ini, saya menukar arah institusi Syafi Coding Academy sebagai pusat persediaan awal untuk membantu pelajar membina asas coding sebelum melangkah ke menara gading.

Hasil Pembelajaran Pelajar

Ringkasan : Pelajar mempunyai asas coding seolah-olah telah berada dalam Semester 1 selama 4 minggu!

Secara terperinci, pelajar bakal :

  • âœ“ Mampu menerangkan "Coding tu apa" dengan yakin
  • âœ“ Mampu membaca setiap baris coding yang ditulis sendiri atau dari Internet/buku
  • âœ“ Mampu menulis coding untuk soalan peringkat asas dari awal hingga habis dengan mudah dan teratur
  • âœ“ Mampu untuk menentukan sama ada code yang ditulis ialah tepat atau tak
  • âœ“ Mampu untuk membaca error dan membaiki error dengan independent (berdikari)

Bagaimana pelajar akan menguasai asas

Sesi 1-to-1 : Hanya anda dan saya, 100% attention saya untuk anda

Untuk setiap topik, saya akan ajar:

  • Theory + gambaran keseluruhan
  • Step-by-step melalui bentuk coding

Setiap pengajaran saya berasaskan :

  • Penerangan secara visual (bentuk lukisan dan diagram)
  • Cara menulis coding dari A sampai Z untuk topik tersebut

Tanya apa jua soalan dan pertanyaan yang terlintas di fikiran

Guided coding session with Sir Syafi (No blind practice)

Sepanjang sesi 1-to-1 nanti, anda akan :

  • Buat coding bersama saya
  • Dapat lihat output sebenar
  • Dapat lihat error sebenar
  • Banyak diam dan berfikir bersama saya (sangat penting dalam coding)

Saya guide anda sehingga siap tetapi bukan dengan menyuap anda jawapan

Saya sentiasa menunggu kata-kata "Tak tahu la sir" untuk saya pimpin anda ke jawapannya.

Saya akan cetus idea anda dan buatkan anda rasa berpuas hati dengan usaha berfikir sendiri
